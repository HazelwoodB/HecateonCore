# Hecateon System Overview

## Platform Vision
Hecateon is the platform identity for deterministic, local-first stability operations. It is designed as infrastructure authority, not a wellness companion metaphor.

## Core Principles
- Local-first operation with encrypted on-device state
- Deterministic and explainable state transitions
- Append-only event and decision history
- Strong operator control and bounded intervention
- Trusted device and recovery-bound access

## Architecture
- Platform: Hecateon
- Runtime: Hecateon Core
- Desktop: Hecateon Desktop
- Modules:
  - Nyphos (Stability Engine)
  - Prometheon (Cognitive Engine)

## Modules
### Nyphos
Owns sleep integrity, mood trend analysis, destabilization risk hysteresis, and intervention ladder outputs.

### Prometheon
Owns Operator/Child/Beast indices, narrative distortion tracking, transaction analysis, and foresight simulations.

## Safety Model
- Explicit escalation boundaries
- Explainable risk outputs
- Deterministic intervention sequencing
- Auditability through immutable event history

## Remote Access Model
- Trusted device registry
- Recovery-code based contingency access
- Profiled launch/operation modes

## Versioning Model
- Identity consolidation release: v2.0.0
