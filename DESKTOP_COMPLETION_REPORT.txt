HOW TO USE
================================================================================

Quick Start (Recommended):
1. Run this command:
   .\RUN_LULLABY_DESKTOP.ps1

2. Wait for both windows to open:
   - Backend console: "https://localhost:5001"
   - Desktop app: Lullaby window

3. Start using:
   - Chat tab: Type message, click Send
   - Health tab: Set mood & sleep, click Log
   - Settings tab: View info & privacy

Manual Start (For Development):
1. Open Terminal 1:
   cd Lullaby\Lullaby
   dotnet run

2. Open Terminal 2:
   cd Lullaby\Lullaby.Desktop
   dotnet run

3. Desktop window opens, connect to backend

Troubleshooting:
- Can't connect? Is backend running on 5001?
- Port in use? Change in Program.cs
- Build fails? Run: dotnet clean && dotnet restore
