HOW TO USE:

1. Launch:
   Double-click: START_LULLABY_DESKTOP.cmd

2. Chat:
   • Type a message
   • Press Ctrl+Enter or click Send
   • See timestamp and reply

3. Health:
   • Click mood emoji (5 options)
   • Adjust sleep slider
   • Click "Log Health Entry"
   • See confirmation

4. Settings:
   • Click "Copy" to copy device ID
   • Click "Show" to reveal recovery code
   • View security features
