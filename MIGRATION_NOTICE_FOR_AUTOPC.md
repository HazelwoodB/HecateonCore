## 📚 What's in HecateonCore?

### **Components:**
- **Hecateon Desktop** - WPF Desktop Application (Primary Interface)
- **Lullaby Web** - ASP.NET Core web dashboard (Secondary)
- **NYPHOS Module** - Wellness tracking (Complete)
- **Prometheon Module** - Cognitive engine (Planned)

### **Features:**
- ✅ Local-first architecture
- ✅ Append-only event store
- ✅ Encrypted storage
- ✅ Sleep/mood/routine tracking
- ✅ Risk assessment engine
- ✅ Crisis intervention UI
- ✅ Multiple launcher systems
- ✅ Comprehensive documentation
